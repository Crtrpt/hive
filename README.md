# hive
物联网设备管理平台

# 功能特性
- 用户管理
    - 角色管理
    - 组织管理
    - 设备分组
- 物模型支持
    - 设备管理物模型支持 
    - 支持通用的协议 
        - mqtt
        - coap
        - http
    - 内建的私有协议支持
    - 私有协议的接入 独立的网关适配 物模型即可

- 设备管理
    - 设备开发维护更新报警 全周期管理
- 平台级联
    - 平台级联树 
    - 设备级联树
- 云云互联
    连接 
    - 阿里云 
    - ctwing
    - onenet
- 场景管理
    - 场景内设备互联互动
